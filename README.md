# Этот бот был сделан в рамках хакатона от VK Education и MAX

## Бот нужен для добавления и просмотра домашнего задания на платформе MAX

### Подразумевается, что бот будет использоваться в групповом чате несколькими участниками, однако в личных сообщениях он тоже работает
### На стадии MVP в бота можно загружать только текст

Предварительно нужно установить IDE (У нас это VS Code), Docker и node.js.

Для того чтобы запустить бота локально нужно:
1) Склонировать репозиторий командой git clone https://github.com/MrSquirel/bot-max.git
2) Установить библиотеки командами:
3) npm i @maxhub/max-bot-api
4) npm i dotenv
5) npm i nodemon
6) npm add -d typescript
7) Далее работаем с Docker, прописываем команду docker init (Docker должен быть запущен) (ЭТО НУЖНО ТОЛЬКО ЕСЛИ ВЫ ХОТИТЕ СОБРАТЬ КОНТЕЙНЕР ЗАНОВО, В РЕПОЗИТОРИИ ОН УЖЕ ЕСТЬ)
8) Выбираем стрелочками и enter Node
9) На вопрос о версии либо уточняем, либо нажимаем enter
10) Далее выбираем npm
11) Команда для запуска npm start
12) Прописываем порт, например 8808
13) Собираем образ командой docker compose up --build 
14) Запускаем бота через Docker, всё должно заработать!

Если нужно запустить бота без Docker, то просто вводим npm start или npm run dev
